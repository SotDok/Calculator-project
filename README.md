# JS Calculator

Simple calculator with mouse & keyboard support.

## Features
- +, -, *, /, decimals
- Clear (`Clear`) and delete (`X`)
- Shows previous operation smaller above result
- Stylish buttons with hover/click effects

## Usage
1. Click or type numbers & operators
2. Press `=` or Enter for result
3. Previous operation shows smaller above current result

Example:

10 + 12
= 22

## Open
Open `index.html` in a browser to use.
